🌟 AI导航站 🌟

本文档整理了我日常使用的AI对话和API服务。

💬 对话模型服务

官方平台

模型 优势 劣势 官方链接
ChatGPT 行业老大（暂时） 严重降智现象，审查严格 openai.com/chatgpt
Claude 降智现象少，回答质量高 封号风险大 anthropic.com/claude
Grok 模型强大，包含thinking模式和deepsearch 暂时仅X订阅用户可用 grok.ai
Google AI Studio 多种模型免费使用，次数限制少，速度快 UI体验差，模型实力一般 aistudio.google.com
Deepseek 模型强大，完全免费 审查严格，上下文窗口小(64k)，服务稳定性不足 deepseek.com
qwen 免费使用，支持中文 模型能力相对较弱 qwen

第三方平台

平台 特点 链接
GitHub Copilot models 稳定，降智概率低，代码能力强 GitHub Copilot models
GenSpark 通过邀请获取多种免费模型使用权，支持deep research genspark.ai
XUB 公益站，模型纯正 xub.ai
Chat01 ChatGPT原生体验，有积分系统，订阅价格较高 chat01.ai
Poe 多种高级模型，但有严格的使用限额 poe.com
Monica 支持Deepseek，但其他高级模型受限 monica.im
秘塔AI搜索 集成DeepseekR1，中文资料搜索优秀 metaso.cn
Aurora AI 免费使用Deepseek，积分/会员使用更多模型 aurorai.cn

🔌 API服务

官方API

服务 特点 链接
Deepseek API 国产大模型，中文支持好 deepseek.com/api
OAIPro 官网API转发，新模型更新快，稳定性高，但价格较贵 oaipro.com
xAI API 目前仅支持Grok-2模型 x.ai/api
OpenRouter 多种模型聚合，需要外卡，有手续费 openrouter.ai
Google AI Studio API 资源丰富，免费额度大 aistudio.google.com
GitHub Models API 代码相关任务表现优秀 github.com/features/copilot
硅基流动 专注国内模型API guijitech.com

服务 特点 链接

PoloAI 专注提供Claude 3.5 Sonnet模型 poloai.top
